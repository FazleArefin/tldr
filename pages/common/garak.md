# garak

> LLM vulnerability scanner.
> More information: <https://docs.garak.ai/garak/>.

- List all available probes/detectors/generators/buffs:

`garak {{--list_probes|--list_detectors|--list_generators|--list_buffs}}`

- Run a quick self-test to verify `garak` installation:

`garak --target_type test.Blank --probes test.Test`
